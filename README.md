A modern dental clinic website with an appointment booking system.

## Features

- Premium landing page with video hero
- Step-based appointment booking flow (Calendly style)
- Email notifications to clinic
- WhatsApp confirmation option
- Backend API built with Node.js + Express

## Tech Stack

- Next.js 15 + Tailwind CSS
- Node.js + Express
- Nodemailer (Email)
- Twilio (WhatsApp, optional)
